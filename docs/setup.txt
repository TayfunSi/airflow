3. Airflow installieren

rm /Users/tsimsek/airflow/airflow.db <- Pfad anpassen
airflow db init



4. User für Web UI anlegen

airflow users create \
    --username admin \
    --password admin \
    --firstname Admin \
    --lastname User \
    --role Admin \
    --email admin@example.com



6. Anpassungen in airflow.cfg 

dags_folder = /Users/tsimsek/airflow/dags <- Pfad anpassen



5. Airflow Webserver und Scheduler starten

Terminal 1: 

airflow webserver --port 8080

Terminal 2:

airflow scheduler



6. Airflow UI öffnen:

http://localhost:8080


7. Airflow stoppen

pkill -f airflow
